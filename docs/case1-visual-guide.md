# 案例1图片版制作指引

## 图片定位

用于网站中展示“个人全平台自媒体运营”的案例拆解图。目标是让HR在10秒内看懂：做了什么、怎么做、结果如何。

## 推荐规格

- 尺寸：1600 x 900
- 风格：蓝白商务风，接近简历视觉
- 工具：Canva / Figma / 稿定设计
- 字数：只放关键词和数据，不放长段落

## 版式结构

### 顶部标题

案例1：全平台自媒体矩阵运营

副标题：
抖音 / 小红书 / 快手内容增长与私域承接

### 左侧：项目背景

- 从0搭建多平台账号矩阵
- 覆盖抖音、小红书、快手
- 目标：内容增长 + 私域转化

### 中间：执行路径

对标拆解 -> 选题策划 -> 脚本打磨 -> 内容发布 -> 数据复盘 -> 私域转化

### 右侧：核心结果

- 抖音粉丝 3.7万
- 小红书粉丝 2.7万
- 快手粉丝 2.4万
- 全网粉丝 10万+
- 总点赞 100万+
- 私域转化 2000+

### 底部：方法沉淀

- 40+对标账号拆解
- 选题库 / 脚本模板
- 爆款内容结构复盘
- 公域到私域转化链路

## 视觉建议

- 不建议放太多装饰元素，优先保证信息清晰。
- 数据区可以用深蓝色大数字，增强可信度。
- 执行路径可以用横向箭头流程图。
- 如果放头像或平台图标，尺寸要小，不要抢案例内容。
